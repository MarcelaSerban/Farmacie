# Gestiune farmacie
# Proiect realizat pentru laboratorul de PIU.

Aplicatie de tip consola (C#) dezvoltata folosind principii POO pentru gestionarea stocurilor unei farmacii.

Structura proiect (clase)
Clasa Medicament: Modeleaza entitatea medicament (atribute: ID, Nume, Producator, Pret, Stoc).
Clasa GestiuneFarmacie: Gestioneaza lista de medicamente si implementeaza operatiile specifice farmaciei.

Functionalitati
Adaugare interactiva a medicamentelor de la tastatura.
Afisare tabelara a listei de obiecte.
Cautare rapida dupa nume.
Stergere conditionata (doar pentru produsele cu stoc 0).
